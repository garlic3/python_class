
Day05 반복문(while), list, tuple, set, dict

while문
	반복 횟수를 모를 때 사용
	while 조건식:
	    실행할 코드
	-> 조건식이 참인 동안 코드가 실행됨

컬렉션
	여러값을 하나의 이름으로 묶은 모음
	list(리스트), tuple(튜플), set(세트), dict(딕셔너리)

	사용목적
	1. 변수명을 여러개 지어야하는 불편함 해소
	2. 값들에게 규칙성을 부여

리스트
	리스트 생성 방법
	1. list명 = [값1, 값2, 값3, 값4...]
	2. list명 = [값]*칸수
	3. list명 = []

	리스트의 길이
	len(list명)
	
	리스트의 사용
		- 추가 
			append(값)
			값을 리스트 맨 마지막에 추가
			insert(인덱스번호, 값)
			값을 해당 인덱스 위치에 추가

		- 삭제
			remove(값)
			값을 리스트에서 삭제
			(단, 중복된 값들이 있을 경우 그중에서 순서가
			 제일 앞에 있는 값을 삭제)
			
			del list[인덱스번호]
			해당 인덱스 번호의 값을 삭제
			
			clear()
			리스트의 값 전체 삭제
			
			pop(인덱스번호)
			인덱스를 전달하면 전달된 인덱스의 값 삭제
			인덱스를 전달하지 않으면 마지막 값을 삭제
		
		- 수정
			list명[인덱스번호] = 값

			
		- 검색
			list명[인덱스번호]
			- list명[-1]:거꾸로 찾기
			- list명[start:end]: start부터 end-1까지

			index(값)
			해당값의 인덱스 반환

		
멤버 연산자
	해당 값의 포함 여부를 판단하는 연산자
	A in B: B에 A가 있으면 참 없으면 거짓
	A not in B: B에 A가 없으면 참 있으면 거짓


튜플(tuple)
	저장된 값을 수정할 수 없는 리스트(추가, 삭제, 수정 불가)
	
	튜플 생성
	1. 튜플명 = (값1, 값2, 값3, 값4...)
	2. 튜플명 = 값1, 값2, 값3....
	
	* 값이 1개인 경우 튜플명 = 값1, 튜플명 = (값1,)

	튜플의 길이
	len(튜플명)
	
	튜플의 사용
		- 검색
		index(값)
		해당 값의 인덱스 반환
		
		튜플명[인덱스번호]
		해당 인덱스의 값 반환

세트(set)

	1. 중복된 값이 저장이 불가능
	2. 저장되는 순서가 없음
	3. 수학의 집합 개념과 유사

	세트의 생성
	1. 세트명 = {값1, 값2, 값3....}
	2. 세트명 = set({})

	추가	
		add(값)
		값을 추가
		
	삭제
		remove(값)
		해당 값이 없을 경우 keyError 발생
		discard(값)
		해당 값이 없어도 오류 발생 x
		pop()
		임의의 값 삭

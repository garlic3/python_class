조건문
	if문
	
	if 조건식1:
	    조건식1이 참이면 실행할 코드
	if 조건식2:
	    조건식2가 참이면 실행할 코드
	if 조건식3:
	    조건식3이 참이면 실행할 코드

	if 조건식1:
	    조건식1이 참이면 실행할 코드
	elif 조건식2:
	    조건식2가 참이면 실행할 코드-> 조건식1이 거짓
	else:
	    위의 조건식이 모두 거짓이면 실행할 코드
	*else는 가장 가까운 if를 인식함

들여쓰기
    스페이스 4개(공백 4개) 또는 탭 1개    

반복문
	for문
	반복되는 횟수를 알거나 알 수 있을 때 사용

	for 변수 in range(초기값, 끝값, 증감값):
	    반복할 코드
	초기값: start => 시작점
	끝값: end => 끝값 -1 까지만 범위 (시작점 <= 변수 < 끝값)
	증감값: step
	
	초기값 생략시 0부터 시작
	증감값 생략시 1씩 증가

기타 제어문
	break
	반복문을 강제로 종료
	
	continue
	반복에서 제외

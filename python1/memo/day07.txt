Day07. 함수, 모듈

다중 반환(return)
	반환값이 여러개 일 경우 하나의 튜플로 반환

기본 매개변수
	기본값(default)이 있는 매개변수
	일반 매개변수 뒤에 위치

가변 매개변수
	매개변수가 여러개 일때 사용
	1. 가변 매개변수
	*args(non-keyworded arguments)
	하나의 튜플로 들어감

	2. 키워드 가변 매개변수
	**kwargs(keyworded arguments)
	하나의 딕셔너리로 들어감

변수의 생명주기
	지역변수
	함수 내부에서 선언한 변수
	함수 안에서만 사용 가능

	전역변수
	함수 밖에서 선언한 변수
	전체 영역에서 사용 가능

모듈
	파이썬 파일
	변수, 함수, 클래스를 모아놓은 파일

	import 방법
	1. from 모듈 import 함수
	2. from 모듈 import 함수1, 함수2
	3. from 모듈 import*
	
	*import 구문은 항상 제일 위에 작성

	별명(alias)
	from 모듈 import 함수 as 별명

	외부 모듈
	외부에서 만들어진 모듈을 불러옴

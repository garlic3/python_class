Day10 파일 입출력

파일 입력
	기존의 파일 내용을 읽어 들이는것

파일 출력
	기존 파일에 새로운 내용을 추가하거나
	새로운 파일을 생성

파일 입출력 방법
	파일 객체 = open(파일명, 모드 + 파일의 종류)
	. : 현재 디렉토리
	.. : 상위 디렉토리

모드
	1. 읽기
		r: read, 파일이 없을 경우 오류 발생
	2. 쓰기
		w: write,  파일이 없을 경우 새로 생성
		(단, 파일이 있을 경우에도 새로 생성)
		a: append, 파일이 없을 경우 새로 생성
		(단, 파일이 있을 경우에는 추가)

		x: exclusive 파일이 없을 경우 새로 생성
		(단, 파일이 있을 경우에는 오류)

파일의 종류
	t: text, 텍스트파일
	b: binary, 텍스트를 사용하지 않는 모든 파일(이미지,사운드,동영상)


파일 읽기
	read(size)
	size: 텍스트 모드일때 글자수, 바이트모드 일때는 바이트수 
	반환값: 읽어들인 문자열, 읽어들인 바이트열
	size 생략시 다 읽어옴
	파일의 끝에 도달하면 빈 문자열("", '')반환

	readline()
	텍스트 파일을 한줄씩 읽어오기
	파일의 끝에 도달하면 빈 문자열("", '')반환

	readlines()
	전체 줄을 모두 읽어 각 줄단위로 리스트에 저장후 리스트를 반환
	만약에 size를 지정하면 해당 바이트 수만큼 읽음 -> 단, 읽어올때 한 행을 완성하는데에
	필요한 만큼 더 읽어옴

csv 파일
	쉼표로 데이터가 분리되어 있는 문서

	읽기
	csv.reader(파일객체, delimiter=",")
	delimiter 구분문자(생략시 기본값은 ,)

	쓰기
	csv.writer(파일객체, delimiter=",")
	delimiter 구분문자(생략시 기본값은 ,)

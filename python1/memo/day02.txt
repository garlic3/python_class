변수 (변하는수 Variable)

	변수는 저장공간
	변수 사용목적: 값을 재사용하기 위해

	변수의 선언(메모리에 할당)
	
	(초기화)
	변수명 = 초기값
	
	변수명: 변수의 이름
	=(대입연산자): 왼쪽에 있는 저장공간에 값을 대입해줌
	값: 데이터

자료형(type)

	정수형	int	1,2,3,4,5....
	실수형	float	1.1, 2.2, 3.3 ... 
	문자열	str	"hello", "안녕", "A", "가".. 
	논리형	bool	True, False 

동적 타이핑
	파이썬은 넣어준 값에 따라서 저장공간(변수)의 타입이 변함
	-> 소스코드가 길어질 경우 예상치 못한 값이 대입될 수 있음

변수 선언시 주의사항
	1. 문자로 시작(숫자로 시작 불가능)
		1b(X) b1(O)
	2. 특수문자를 사용할 수 없음(단, _언더바는 사용가능)
	3. 공백을 사용할 수 없음
		my name(x) 
		my_name(O) -> 파이썬에서 주로 사용
			   -> 팟홀 표기법 또는 스네이크 표기법
		myName(O)  -> 카멜 표기법
	4. 소문자를 사용(특수한 상황에 대문자를 사용)
	5. 뜻이 있는 단어를 선택
		a = "서민성" name = "서민성"
	6. 한글은 사용하지 않음
	7. 키워드 사용 하지 않음
		ex) class(x) if(x) while(x)


제어문자
	1) 이스케이프 문자
		특수한 의미를 가진 문자 \(역슬래시) -> 문자열 안에서 사용!
		\n	줄바꿈
		\t	탭
		\'	작은 따옴표
		\"	큰 따옴표
		
	2) 인터프리터 해석방향
		위에서 아래로
		왼쪽에서 오른쪽으로


	3) print("문자열1", end="문자열2")
		문자열1을 다 출력 한 후 끝에 문자열2가 출력됨
		print()에는 end="\n"가 생략되어 있음


서식문자
	1) % 포맷팅
		"문자열" % 값

	%s	string 문자열	
	%d	decimal 정수(10진수)
	%f	float 실수
	%c	character 문자
	%o	octal 정수(8진수)
	%x	hexadecimal 정수(16진수)
	%%	%
	
		
	* 자릿수를 지정할 때
	정수: Nd
	실수: 0.Nf

	* 여러개를 사용할 때
	print("%d%d" %(2,3))


	2) format 함수
		-> 문자열값.format()
	
		(1) 값을 직접 대입
			print("문자열{} 문자열".format(값))
		(2) 변수로 대입
			-> 중괄호 위치에 순차적으로 대입
			-> 프로그래밍 언어에서는 0번째가 첫번째 자리
			print("문자열{} 문자열{}".format(값1, 값2))

		(3) 키워드(이름)으로 대입
			print("오늘 날씨는 {키워드1}입니다. 계절은 {키워드2} 입니다".format(키워드1="문자열1", 키워드2="문자열2")


	3) f-string
		python 3.6이상 부터 사용 가능
		%포맷팅과 format()함수보다 속도가 빠름
		print(f"문자열 {변수명} 문자열 {변수명}")


입력함수
	input(문자열)
		사용자가 입력한 값을 문자열(str)타입으로 전달
		가져온 값을 변수에 담아서 전달
		-> 괄호 사이에 문자열이 있다면 입력받을때 출력됨
		-> 입력함수로 사용자로부터 값을 입력받고 필요한 타입으로 형변환

자료형 변환(casting)

	1. 자동 형변환
		정수 + 정수 = 정수
		정수 + 실수 = 실수
	2. 강제 형변환
		-> 원하는 자료형으로 변환(단, 변환할 수 없는 값을 넣을 경우 에러발생)
		int()
			int(정수문자열)
			int(실수)
		float()		
			float(실수문자열)
			float(정수)
		str()
			str(정수)
			str(실수)
